Introduction

How Aqua Sort Works

  1. Select a tube that has color on top.
  2. Select another tube to pour into it.
  3. You can only pour into an empty tube or matching top color.
  4. Win when every non-empty tube has one color only.

Tip: keep at least one empty tube open so you can reorder colors more freely.
